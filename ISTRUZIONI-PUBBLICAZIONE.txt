N•E•B•R•A v0.9.93 — PUBBLICAZIONE SU GITHUB PAGES

1. Estrai lo ZIP.
2. Copia tutti i file nella radice del repository GitHub.
3. In GitHub apri Settings > Pages.
4. Seleziona Deploy from a branch, il branch principale e la cartella / (root).
5. Salva e attendi la pubblicazione.

La PWA richiede HTTPS, fornito automaticamente da GitHub Pages.
Non rinominare index.html, manifest.webmanifest, sw.js o le icone.
