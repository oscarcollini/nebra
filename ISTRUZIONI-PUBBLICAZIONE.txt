N•E•B•R•A v0.9.93 — PUBBLICAZIONE SU GITHUB PAGES

1. Estrai tutti i file dello ZIP nella radice del repository.
2. Verifica che `index.html` sia nella radice, non dentro una sottocartella.
3. In GitHub: Settings > Pages > Deploy from a branch > main > /(root).
4. Apri l'indirizzo HTTPS pubblicato.
5. Dopo il primo caricamento online, prova installazione e ricarica offline.

Il service worker richiede HTTPS oppure localhost; non funziona aprendo `index.html` come file locale.
